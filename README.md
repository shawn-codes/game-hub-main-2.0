# GameHub2.0

GameHub is a comprehensive video game discovery web application designed to help gamers explore and find new and exciting titles to play. Whether you're a casual player or a dedicated enthusiast, GameHub makes it easy to search for games tailored to your preferences. You can filter your search by platform, genre, release date, and even specific features, ensuring you discover games that match your interests.

## Getting Started

To get started with GameHub, follow these steps:

1. Clone this repository to your local machine.
2. Run `npm install` to install the required dependencies.
3. Get a RAWG API key at https://rawg.io/apidocs. You'll have to create an account first.
4. Add the API key to **src/services/api-client.ts**
5. Run `npm run dev` to start the web server.
